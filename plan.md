## 1. 本次更新：仙侠故事专栏 📖

### 1.1 新增模块：「穹顶之下」有声可视仙侠读物

**背景**：在仙侠世界的奇幻背景下，融入学习与成长的主题，用诙谐幽默的方式向学生和职场年轻人弘扬正确的世界观和积极向上的正能量。

### 1.2 目标用户

| 群体 | 特征 |
|------|------|
| 学生 | 初高中/大学，碎片化阅读习惯，喜欢轻松搞笑内容 |
| 职场新人 | 通勤场景，需解压、治愈内容，偏好仙侠/二次元 |

## 2. 功能规划

### 2.1 故事专栏入口

| 功能 | 说明 |
|------|------|
| 底部导航新增「故事」Tab | 与「岛屿」「换装」等 Tab 并列 |
| 章节列表页 | 展示所有章节，封面图 + 标题 + 简介 |
| 阅读器页 | 沉浸式阅读，支持图文排版 |

### 2.2 内容形式

| 形式 | 说明 |
|------|------|
| 动态漫画 | 图片 + 对话气泡 + 分镜叙事 |
| 有声朗读 | 配音 + 动效同步，支持暂停/播放 |
| 阅读进度 | 自动记录，支持书签定位 |

### 2.3 第一话规划

| 项目 | 内容 |
|------|------|
| 标题 | 初入仙门·第一话「天上掉下个大师兄」 |
| 调性 | 轻松搞笑，成长共鸣 |
| 时长 | 约 3 分钟阅读 |
| 主题 | 主角误入仙门，笨拙但努力地适应新环境 |

## 4. 技术方案

### 4.1 前端

| 层 | 技术 | 说明 |
|----|------|------|
| 框架 | React | 故事 Tab、阅读器 |
| 样式 | CSS | 响应式排版、播放器样式 |
| 音频 | HTMLAudioElement | 配音播放控制 |
| 状态 | React State / localStorage | 阅读进度、书签 |

### 4.2 数据模型

```typescript
interface Story {
  id: string;
  title: string;
  description: string;
  coverImage: string;
  chapters: Chapter[];
}

interface Chapter {
  id: string;
  title: string;
  content: ContentBlock[];
  audioUrl?: string;
  duration: number; // 分钟
}

interface ContentBlock {
  type: 'image' | 'text' | 'dialog';
  content: string;
  imageUrl?: string;
}

interface ReadingProgress {
  chapterId: string;
  progress: number; // 0-100
  completed: boolean;
  lastReadAt: string;
}
```
