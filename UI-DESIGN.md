## 1. 设计理念

**"云海之上，仙侠人间"**

以蛋仔派对的休闲漫游感为骨架，以仙侠古风为皮。画面追求"可静的仙侠，可动的游戏"——既有云雾飘渺的意境，又有 Q 版角色的活泼跳脱。

核心体验关键词：**仙气、圆润、漂浮、古风**。

## 5. 组件规范

### 5.1 底部 Tab 栏

```
结构：
┌──────────────────────────────────────────┐
│  ☐         ☐        ☐       ☐       ☐│
│  岛屿      换装      背包     好友     设置│
└──────────────────────────────────────────┘

样式：
- 背景：#1A1A2E 透明度 90% + 顶部1px渐变描边
- 圆角：24pt（整体胶囊形）
- 阴影：黑色 40% 透明度，y偏移5pt，模糊15pt
- 距屏幕底：20pt（安全区外）

状态：
- 未选中：图标 #F5E6C8 50%，文字 #F5E6C8 40%
- 选中：图标 #E85D75，文字 #F5E6C8 100%
- 切换动画：spring(response 0.3, damping 0.7)
```

### 5.2 虚拟摇杆

```
结构：
┌────────────┐
│    ○      │  ← 旋钮（渐变填充）
│  ┌──────┐ │  ← 底座（半透明+描边）
│  │      │ │
│  └──────┘ │
└────────────┘

样式：
- 底座：110x110pt，#7B5EA7 透明度 30%，描边 #F5E6C8 40%
- 旋钮：44x44pt，径向渐变（#E85D75 → #7B5EA7）
- 阴影：#E85D75 透明度 50%，模糊8pt

位置：距左40pt，距Tab栏顶100pt（屏幕左下）

交互：
- 按住旋钮拖动，限制在底座半径45pt内
- 松开后旋钮 spring 回中心
- 回弹动画：spring(response 0.15)
```

### 5.3 卡片组件

```
样式：
- 背景：#7B5EA7 透明度 15%
- 描边：#7B5EA7 透明度 30%，1px
- 圆角：12pt
- 内边距：16pt
```

### 5.4 按钮

```
主按钮：
- 背景：#E85D75
- 文字：#F5E6C8，Bold 15pt
- 圆角：12pt
- 内边距：12pt 垂直，20pt 水平
- 阴影：#E85D75 透明度 30%，模糊6pt

次按钮：
- 背景：#7B5EA7 透明度 30%
- 描边：#7B5EA7 100%
- 文字：#F5E6C8 80%
- 圆角：12pt

禁止态：
- 整体透明度 50%
- 触摸事件禁用
```

### 5.5 颜色选择器

```
布局：6列网格，间距12pt
每个色块：40x40pt 圆形

状态：
- 默认：纯色填充，无描边
- 选中：白色描边3px + 外发光（#E85D75 60% 透明度，模糊6pt）
- 触摸反馈：scale 0.92，持续100ms
```

## 6. 岛屿场景设计

### 6.1 场景层次（Z 轴排序）

| 层 | Z Position | 内容 |
|----|-----------|------|
| 星空背景 | 0 | 闪烁星星粒子 |
| 云海层 | 10 | 底部云雾粒子 |
| 远景浮岛 | 20 | 顶部小浮岛（天宫岛） |
| 中景浮岛 | 30 | 左右两个子浮岛 |
| 主浮岛 | 40 | 仙侠主岛 + 建筑 |
| 角色层 | 100 | Q 版仙侠角色 |
| UI 层 | 200 | 虚拟摇杆、状态标签 |

### 6.2 浮岛设计

**仙侠主岛**
- 尺寸：300x80pt
- 颜色：#3D2B1F（岛岩棕）
- 描边：#7B5EA7 3px
- 动画：上下悬浮 ±8pt，周期 5s，ease-in-out

**灵草岛（左）**
- 尺寸：180x60pt
- 颜色：#2D4A2D
- 标注："灵草岛"

**飞剑岛（右）**
- 尺寸：200x70pt
- 颜色：#1F3D5C
- 含传送门、瀑布

**天宫岛（上）**
- 尺寸：140x50pt
- 颜色：#4A1F4A

### 6.3 建筑节点

**亭子**
```
组成部分：
- 底座：60x8pt，#7B5EA7
- 柱子：4x30pt x4，#F5E6C8，均匀分布
- 屋顶：80x12pt，#E85D75，圆角3pt
- 宝顶：圆形 r=5pt，#F5E6C8
```

**楼阁**
```
组成部分：
- 底层：50x40pt，#7B5EA7，描边 #F5E6C8
- 上层：36x35pt，#7B5EA7，描边 #F5E6C8
- 塔顶：20x25pt，#E85D75
```

### 6.4 角色精灵

```
整体比例：Q 版，头占身体 1/2

组成：
- 头：圆形 r=20pt，#F5E6C8，描边 #7B5EA7 2px
- 头发：26x16pt，圆角6pt，#1A1A2E
- 发髻：圆形 r=6pt，#E85D75
- 身体：30x35pt，圆角8pt，白色，描边 #7B5EA7 2px
- 披风：20x25pt，圆角4pt，#E85D75 80%透明度

动画：
- 待机：上下呼吸 ±5pt，周期 2400ms
- 行走：跳跃 ±8pt，周期 300ms
- 披风：待机静止，行走时加速左右摆动
```

### 6.5 特效设计

**云海粒子**
- 颜色：#F5E6C8
- 透明度：60%，速度衰减至 30%
- 尺寸：0.3，增长 0.2
- 位置：场景底部 y=-300

**仙鹤**
- 身体：椭圆 20x12pt，白色
- 头：圆形 r=6pt，白色
- 喙：矩形 8x3pt，#F5E6C8
- 翅膀：椭圆 15x8pt，#1A1A2E
- 动画：斜向上飞行路径，翅膀拍打

**传送门**
- 圆形 r=25pt
- 填充：#7B5EA7 30%透明度
- 描边：#E85D75，3px
- 动画：脉冲透明度 0.3↔1.0，800ms；持续旋转，4000ms/圈

## 8. 动效规范

### 8.1 全局动效曲线

| 用途 | 曲线 | 参数 |
|------|------|------|
| Tab 切换 | spring | response 0.3, damping 0.7 |
| 摇杆回弹 | spring | response 0.15 |
| 页面过渡 | easeInOut | duration 0.3 |
| 按钮按压 | easeOut | duration 0.1 |
| 卡片出现 | easeOut | duration 0.25 |

### 8.2 SpriteKit 动画

| 动画 | 类型 | 参数 |
|------|------|------|
| 浮岛悬浮 | moveBy Y±8 | duration 2.5，repeat forever |
| 星星闪烁 | fadeAlpha 1↔0.2 | duration 1-3s 随机，repeat forever |
| 角色呼吸 | moveBy Y±5 | duration 1.2，repeat forever |
| 角色行走 | moveBy Y±8 | duration 0.15，repeat forever |
| 仙鹤飞行 | moveBy + rotate | duration 3s，repeat forever |
| 翅膀拍打 | rotate ±0.3 | duration 0.3，repeat forever |
| 传送门旋转 | rotate 360° | duration 4s，repeat forever |
| 传送门脉冲 | fadeAlpha 0.3↔1.0 | duration 0.8，repeat forever |
